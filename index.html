Giới thiệu

Mình là một kỹ sư phần mềm đam mê công nghệ trí tuệ nhân tạo và tự động hóa quy trình. Mình tập trung vào việc đưa mô hình ngôn ngữ lớn (LLM) từ lý thuyết vào ứng dụng thực tế hàng ngày, đặc biệt là xây dựng các giải pháp Agent hỗ trợ lập trình viên tối ưu hóa 200% năng suất làm việc.

Với triết lý tối giản (**KISS**), hiệu quả (**DRY**) và thực tiễn, mình xây dựng những sản phẩm có giá trị sử dụng cao, giao diện mượt mà và hệ thống ổn định.

AI Workspace Của Mình

Hạ tầng chạy ngầm phục vụ phát triển ứng dụng thông minh.

Intel NUC Server

Máy chủ dịch vụ trung tâm chạy Docker 24/7 để vận hành các MCP (Model Context Protocol) và là hub định tuyến LLM chính.

  • 9Router (Port 20128)
  • Local LLMs Inference
  • MCP Server Hub

9Router (Proxy & Load Balancer)

Giải pháp proxy tích hợp combo model thông minh (SPARK) tự động xoay vòng và fallback khi gặp lỗi/hạn mức rate-limit.

  • Auto-Fallback 3 cấp độ
  • Quota Tracking thời gian thực
  • Gộp nhóm model (Combo pack)

OpenClaw Gateway

Kết nối các AI Coding Agent từ máy cá nhân (Mac mini) trực tiếp đến máy chủ 9Router trên NUC để lập trình tự động hóa cực nhanh.

  • Tự động hóa CLI chat
  • Custom Skills & Playbooks
  • An toàn Command Allowlist
